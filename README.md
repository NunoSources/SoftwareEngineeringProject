## Instructions to Execute
Execute Node JS:

(BackEnd) Port: 3080

Go to backend/ directory, open a terminal and type:
```
npm i
```
```
DEBUG=backend:* npm run devstart
```

Execute Angular

(FrontEnd) Port: 3030

Go to frontend/ directory, open a terminal and type:
```
npm i
```
```
ng serve -o
```
